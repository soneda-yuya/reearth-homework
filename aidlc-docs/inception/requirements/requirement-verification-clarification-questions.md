## 曖昧点 1: reearth-cms の初期セットアップ範囲

**質問 8 の回答**: `X) integration tokenを手動で取得するので、それを元に他は作成して欲しい`

reearth-cms では、Integration Token は **Workspace** に紐付き、そこから **Project → Model → Field** の階層があります。Integration API でモデル・フィールドの作成ができるか、そもそもプロジェクトレベルの作成はUI必須かで手順が変わるため、どこまで手動・どこから自動化するかを明確化させてください。
